# 🎯 小妹学英语 (Sister Learning App) 网页版进阶任务书

### feat: 互动体验与视觉重塑 (High-Feedback UI)

**引用上下文**
请参考：**@index.html**、**@style.css**、**@script.js** 和 **@words_list.csv**。

**提示词 (Prompt)**

根据当前的界面原型，进行深度的功能修复与视觉重塑，目标是打造一个“高情感反馈、低压力、童真感”的交互体验。

#### 1. 核心功能修复（最高优先级）
* **语音朗读**：检查并在 `script.js` 中修复喇叭图标的点击事件。点击时，必须调用浏览器的 `Web Speech API` (使用 `speechSynthesis`) 朗读当前英文单词。确保逻辑正确，无报错。

#### 2. 高光时刻：正确反馈系统 
在“今日学习”界面，当用户输入拼写正确并点击提交时：
* **视觉高潮**：使用原生 CSS 动画在屏幕上生成并弹出一堆升起的虚拟气球（Balloons）或旋转的星星（Stars），持续 2 秒后消失。
* **听觉鼓励**：同步调用 Web Speech API，让 AI 用温柔的声音说：“太棒了！” 或 “正确！”。
* **积分反馈**：总积分（total_points）数字要有一个轻微的放大缩小抖动动画，表示积分增加。

#### 3. 视觉体系重塑 (视觉一致性)
参考 **@README.md** 中的配色：
* **配色方案**：全站背景改为 `#FAFAFA`。侧边栏保持蓝色调但需更柔和。单词卡片使用纯白背景 `#FFFFFF`。
* **圆角与阴影**：所有的卡片、按钮、输入框、侧边栏都必须加上 **`border-radius: 15px;`** 以上的大圆角，并添加微妙的 **`box-shadow`**（如 `0 4px 6px rgba(0,0,0,0.1)`），使其看起来像悬浮的魔法卡片。
* **字体**：设定全站使用无衬线、偏圆润的系统字体（如 `system-ui, -apple-system, sans-serif`）。

#### 4. 界面布局重构 (UI Redesign)
* **侧边栏**：
    - 增加 padding，让内容不要太拥挤。
    - **宠物可视化**：将“宠物状态”里的 Emoji 蛋放大到原来的 2 倍，居中显示，并加上一个简单的“呼吸（Pulse）”CSS 动画。
* **单词卡片区域**：
    - 将整个区域包裹在一个带有圆角和阴影的、悬浮感强的 `div` 卡片中。
    - **单词展示**：放大单词 font-size，调整音标、中文释义的层级和间距。喇叭图标要和单词放在同一行，大小要适中。
    - **输入框**：输入框要设计成圆角矩形，有明显的 Border 提示聚焦。
    - **按钮组**：提交按钮和下一个按钮要使用圆角、有 Hover 效果的扁平化设计，使用主题色。

**添加规则**
* 必须使用纯原生技术实现（HTML/CSS/JS）。
* 鼓励反馈的 CSS 动画代码请直接内嵌在 `style.css` 中。
* 严禁修改已有的数据加载逻辑。

**Commit**: `feat: 修复语音朗读并重塑高反馈、童真感 UI`
