# sister_learning_app
这是一个帮助小学生学习英语的软件，主要功能包括：
1. 提供英语单词的学习功能，包括单词的记忆、拼写和理解。
2. 提供英语句子的学习功能，包括句子的记忆、语法和理解。
3. 提供英语对话的学习功能，包括对话的记忆、语法和理解。
## 特点
1. 界面简单易用，适合小学生使用。
2. 提供丰富的学习内容，包括单词、句子和对话。
3. 支持个性化学习，用户可以根据自己的学习进度和需求进行调整。
## 主要学习功能：
1. 🔤 单词全能王：涵盖记忆、精准拼写及多维度语义理解，夯实词汇基础。
2. 📝 句法实验室：通过经典句型拆解，同步训练语法逻辑与语感。
3. 💬 情境模拟对话：模拟真实生活场景，提供可追溯的对话练习。
4. 📖 故事剧场 (特色功能)：
5. 情境串联：将孤立单词编织成生动的小故事，实现“词不离句，句不离篇” 。
6. 交互式测评：配套理解性题目，强化长效记忆。
7. 📈 成长足迹：自动跟踪学习进度，量化单词、句子及对话的掌握数量 。
8. 积分系统：用户完成学习任务后，会获得积分，积分可以用来购买萌宠的商品，也可以用来喂养萌宠。
## 激励机制
1. 学习奖励：用户完成每日学习任务后，会获得积分奖励，如学习10个单词，就会获得10分。
2. 陪伴学习萌宠：用户刚开始学习时，可以选择学习一个萌宠，如狗或猫，与它们互动，增加学习的趣味性。
3. 积分喂养萌宠：用户可以通过完成学习任务获得积分，积分可以用来喂养萌宠，如给萌宠喂食或训练萌宠。例如喂养一百积分后宠物会从蛋孵化成小狗，再喂养一百积分后小狗能够实现基本的互动。再100积分后小狗可以进行互动评论，形状也会变大一些
4. 积分奖励：用户可以通过完成学习任务获得积分，积分可以用来购买萌宠的商品，如给萌宠买衣服或喂萌宠。
5. 总共达到有600积分的时候，小狗会说：我可以完成你一个心愿，用户可以输入心愿，输入完成后小狗说我将完成你的心愿。这个心愿是由家长来完成的
## UI设计：
1. 主界面：简洁明了，包含学习单词、句子、对话和故事理解性学习的选项。
2. 学习单词界面：显示单词、拼写和理解的选项，用户可以根据需要进行学习。
3. 学习句子界面：显示句子、语法和理解的选项，用户可以根据需要进行学习。
4. 学习对话界面：显示对话、语法和理解的选项，用户可以根据需要进行学习。
5. 故事理解性学习界面：显示单词组成的小故事和题目，用户需要根据理解的内容填写答案。
6. 学习进度跟踪：用户可以查看自己的学习进度，包括学习的单词、句子和对话的数量。
7. 色彩设计：界面采用简单的颜色组合，如蓝色、白色和黄色，以提高可读性和用户体验。
8. 童真感：界面设计注重儿童友好，采用简单的图标和颜色，以减少学习压力和提高用户满意度。
本项目致力于创造一个“低压力、高反馈”的儿童友好型交互环境 ：
色彩心理学：采用蓝色（理智）、白色（简洁）与黄色（活泼）的科学组合，提升可读性并缓解视觉疲劳。
视觉一致性：使用童真感十足的图标与扁平化设计，确保主界面、学习页与故事页的风格统一。
极简交互：针对小学生操作习惯，简化层级结构，实现“一点即学”。
## 安装和运行
1. 确保您的计算机上安装了Python 3.x版本。
2. 克隆此仓库到您的本地计算机：`git clone https://github.com/yourusername/sister_learning_app.git`
3. 进入项目目录：`cd sister_learning_app`
4. 安装依赖项：`pip install -r requirements.txt`
5. 运行应用程序：`streamlit run app.py`
